RESUME UPLOADS FOLDER
===================

This folder is for storing your resume PDF files that visitors can download.

Instructions:
1. Add your resume files here with these exact names:
   - Shivpujan_Kumar_Resume.pdf (Main resume - 1-2 pages)
   - Shivpujan_Kumar_CV.pdf (Full CV - 2-3 pages)
   - Shivpujan_Kumar_Technical_Resume.pdf (Technical skills focused - 1 page)

2. Files should be in PDF format for best compatibility

3. Place PDF files directly in this folder (no subfolders)

4. The portfolio will automatically display these files with download buttons

5. Update the Resume.tsx component if you want to change file names or descriptions

Example file structure:
public/
├── resume/
│   ├── Shivpujan_Kumar_Resume.pdf
│   ├── Shivpujan_Kumar_CV.pdf
│   └── Shivpujan_Kumar_Technical_Resume.pdf

Note: You can add, remove, or modify the resumeFiles array in src/components/Resume.tsx 
to customize which documents are displayed.
